# 情绪氛围生成器 — 设计文档

## 概述

一款纯前端情绪氛围生成器。用户选择当下心情（放松、emo、热血、发呆），页面自动切换渐变背景、动态粒子动画、轻音乐和简约标语。支持自定义文字和配色，设置通过 localStorage 持久化。可当作桌面氛围页使用。

## 技术栈

- 纯前端：HTML + CSS + 原生 JavaScript（ES6+）
- Canvas 2D 粒子系统（requestAnimationFrame）
- HTML5 `<audio>` 标签
- IndexedDB 存储用户上传的 MP3 文件
- localStorage 存储用户设置
- 无构建工具，无第三方依赖

- 自定义按钮：右下角 ⚙ 图标

## IndexedDB 存储

- 数据库名：`MoodAtmosphereDB`
- 对象存储：`audioFiles`
- key：氛围 ID（`relax` / `emo` / `firedUp` / `zoneOut`）
- value：`{ id, fileName, blob }`
- 用户上传 MP3 → 存为 Blob → 页面加载时读取 → 生成 Blob URL 供 `<audio>` 使用

## 粒子系统

- Canvas 2D，全屏分辨率（监听 resize 事件调整）
- requestAnimationFrame 驱动，目标 60fps
- 每个氛围预设定义粒子参数：
  - `shape`: `"circle"` | `"line"` | `"polygon"`
  - `speed`: 0.2~2.0（基础速度）
  - `direction`: `"up"` | `"down"` | `"random"`
  - `colorRange`: [开始色, 结束色]（粒子颜色在此范围内随机）
  - `density`: 10~100（每 10,000px² 的粒子数基准）
- 切换氛围时：旧粒子逐渐减少 + 新粒子逐渐增加，过渡 1.5s
- 性能优化：粒子数上限 200，低端设备可降低密度

## 过渡动画

| 元素       | 过渡方式                       | 时长  |
|------------|-------------------------------|-------|
| 渐变背景   | CSS transition background     | 1.5s  |
| 粒子       | JS 渐进替换（旧淡出 + 新淡入）  | 1.5s  |
| 标语       | CSS scale(0.9→1) + opacity    | 0.6s  |
| 音乐       | JS 交叉淡入淡出（音量渐变）     | 2.0s  |

## 边界情况处理

- **音乐加载失败**：静默降级，显示 🔇 图标，不阻断其他功能
- **Canvas 不支持**：降级为纯 CSS 渐变背景，移除粒子层
- **IndexedDB 不可用**：上传功能降级为仅本次生效（使用 Blob URL，刷新丢失），并提示用户
- **localStorage 不可用**：所有自定义仅本次生效，不报错
- **用户上传非 MP3 文件**：前端校验文件类型，显示提示
- **用户上传超大文件**：限制 15MB，超出提示
- **移动端**：响应式布局，按钮适配触控尺寸（≥44px），粒子密度自动降至 50%
